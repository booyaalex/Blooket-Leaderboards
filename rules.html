Submission Rules

  1. Refrain from using profanity in your display names.
  2. Do not create multiple display names or leaderboard spots for one blooket account.
  3. Do not put multiple of your own blooket accounts on the leaderboard.
  4. Only use the characters A-Z, a-z, and 0-9 when creating display names.

General Run Rules

Record The Full Run

When recording, you must record from the very latest the start point. The start point is defined as:

  1. When the game asks you if you want to use a tutorial. The timer will start when you press either Yes or No to the tutorial.
  2. When the start button is pressed on the host screen. The timer will start once the Start button is pressed.
The run ends the moment you complete the given task of the run in a speedrun, or when you end the recording when not doing a speedrun.


Speed Up

You are not allowed to speed up any video that is shorter that 15 minutes long. If you speed up a video, you must provide how much you sped up the video and how long the original video was.


Spicing

You may not edit the video in any way, the main form of editing being splicing. Splicing is defined as cutting out parts of the video.


Glitch Rules

The use of glitches in runs are not allowed unless stated otherwise. A list of glithces will be provided below. Remember that this list may be updated when new glitches are discovered.


Flappy Blook

Pausing

Pausing is defined as changing your blook during the run, causing the game to pause while keeping your current score.


Tower Defense

Save Scumming

Save Scumming is defined as reseting a round until you get the desired result, usually completing the round.


Stacking

Stacking is defined as multiple towers being placed on one singular game tile.
